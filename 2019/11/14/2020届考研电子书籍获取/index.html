前言

考研复习的第一步中，大多人会纠结用什么辅导书，而辅导书又比较贵，一般全书都要50元以上，不可能一本一本都买来试看，因此先大致看看电子档，确定一下自己是否喜欢这本书比较重要。

考研书籍选择原则

一般从以下几个角度来看：

  • 优先选择自己听的视频课老师出版的电子书籍；
  • 该书籍往年学长学姐的评价（我的教程里有一定推荐，另外多看看知乎的评价）；
  • 书籍的排版、格式、例题解析仔细程度、习题数量、知识点总结是否丰富等；

总结

上述基本列出了市面上大部分的考研名师所出版的辅导资料，相信同学们已经有点眼花缭乱了。不过不要紧，反正电子档已经给学弟学妹们收集好了，大可花个大半天时间下载下来看，按我上面说到的挑选原则来挑选一下，选择适合自己，能够从一而终的辅导书，考研需要的是一心一意，不是朝三暮四。如果同学实在纠结，那么我的建议如下（仅是建议，不代表最优选择）：

数学：
  • 张宇高数18讲 + 李永乐线性代数辅导讲义 + 张宇概率9讲（喜欢张宇老师讲课风格可选这套）
  • 李王全书（不打算听张宇课程可选这套）
  • 李林880题+108题 / 李永乐660 + 李林108 + 张宇闭关修炼
  • 张宇真题大全解
  • 李林冲刺6套卷 + 李林冲刺4套卷 + 张宇最后4套卷
英语：
  • 恋恋有词 + 唐静拆分组和翻译法
  • 唐迟阅读的逻辑
  • 朱伟写作考前冲刺小书
  • 黄皮书历年真题解析
政治：
  • 肖秀荣精讲精练
  • 肖秀荣1000题
  • 腿姐考研政治冲刺背诵手册 or 徐涛考研政治冲刺背诵笔记
  • 腿姐十年真题试卷
  • 模拟题必做系列(优先级从前往后)：肖四、腿姐4套卷、徐涛8套卷、肖八、蒋中挺5套卷。模拟题使用说明参考我的博客：考研政治复习攻略。
